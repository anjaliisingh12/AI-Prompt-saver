# 🚀 AI Prompt Saver

AI Prompt Saver is a MERN stack web application that allows users to save, manage, and organize their AI prompts in one place.  
It helps users quickly access, reuse, and manage prompts efficiently.

🌐 **Live Demo:**  
👉 https://ai-prompt-saver-six.vercel.app/

---

## 📌 Features

- 🔐 User Authentication (Login / Register)
- ➕ Add New Prompts
- ✏️ Edit Saved Prompts
- ❌ Delete Prompts
- 📂 Organized Prompt Management
- ⚡ Fast and Responsive UI

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/anjaliisingh12/AI-Prompt-Saver.git
```

### 2️⃣ Install dependencies

For backend:

```bash
cd server
npm install
```

For frontend:

```bash
cd client
npm install
```

### 3️⃣ Add Environment Variables

Create a `.env` file inside the server folder and add:

```
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

### 4️⃣ Run the project

Backend:

```bash
npm start
```

Frontend:

```bash
npm start
```
